# AndroidControl
这是一款实时控制Android手机的工具

### 包含内容

* 网页端😀
* 客户端😀
* 服务端😀


![](demo.gif)

### 使用方法

1. 使用Intellij Idea打开项目并运行
2. 打开`web/index.html`，并传入参数 `index.html?ip=127.0.0.1&port=6655`

注意：

* 测试的手机需要打开Debug模式

### 日后
* 帧数限制，用户可自定义帧数
* 画面清晰度，旋转，缩放设定
* 群控功能，可以控制多台设备
* 操作映射，对一台机器的操作映射到多台机器
* 操作录制(脚本录制)
* 服务器支持
* 流压缩
